# **App Name**: PagePilot

## Core Features:

- PDF Upload: Accepts a PDF file via drag-and-drop or file input.
- Flipbook Display: Renders PDF pages as interactive flipbook pages.
- Smooth Animation: Uses smooth, realistic page-turning animations (React-Pageflip or similar).
- Page Navigation: Offers forward/backward navigation (click, drag, or arrow buttons).
- Dynamic Page Generation: Generates pages dynamically from uploaded PDF, ensuring support for multiple pages.
- PDF Download: Downloads the original PDF file.
- Dark/Light Mode: Toggles between a light and dark color scheme for improved user experience.

## Style Guidelines:

- Primary color: Deep blue (#2E5BFF) to evoke sophistication and digital reliability.
- Background color: Very light blue (#E5EAF8), close to white, for a clean backdrop that emphasizes the flipbook.
- Accent color: Purple (#C669FF) for interactive elements.
- Body and headline font: 'Inter' sans-serif for a modern, neutral, readable style.
- Use clean, outline-style icons for navigation and controls.
- Book-like layout with realistic page shadows and subtle 3D effects to enhance the flipbook appearance.
- Implement smooth page transitions and subtle hover effects for an engaging user experience.
